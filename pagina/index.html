Crear un cilindro en AutoCad 3D

En esta pagina se mostrara el paso a paso para crear un cilindro en AutoCad 3D. Ademas de cilindros se pueden hacer otras formas. jaja

¿Que es AutoCad?

AutoCAD es un programa de ingeniería asistido por computadora que se enfoca en el diseño de elementos en dos dimensiones (2D) y tres dimensiones (3D). Este software, desarrollado por la empresa Autodesk, permite dibujar y modelar elementos en ingeniería. Su interfaz amigable y llena de herramientas facilita la creación de diseños, y su potencia lo convierte en un referente importante en la industria del diseño, manufactura y producción. Desde simples líneas hasta el modelado de estructuras complejas, AutoCAD es una herramienta esencial para arquitectos, ingenieros y diseñadores1. 🏗️🖥️

Ajustar mesa de trabajo

Nuevo Dibujo

Una vez abierto AutoCad, en la parte superior izquierda como se muestra en la imagen, se le deb dar click a la "cruz" para crear una nueva mesa de trabajo.

Seleccionar opciones 3D

Autocad esta configurado para iniciar con las herramientas 2D una vez abierto el programa, para cambiar esta opcion, en la parte inferior derecha se le debe dar click al boton de engranaje, como se muestra en la imagen.

Opción 3D Basics

Una vez hecho lo anterior, se desplegaran varias opciones, como se trabajara un modelo basico de diseño, se escogera la opcion que se muestra.

Pasos para crear el cilindro

Formas

En la parte superior izquierda se mostrará como primera instancia un cubo, si le damos click a este se desplegará una lista con todas las formas posibles.

Cilindro

Dentro de la lista, seleccionar la opcion "cylinder".

Radio

Dentro de la mesa de trabajo, dar click derecho para ubicar el cilindro luego de esto nos pedira colocar el radio de este. como las unidades vienen por defecto en metros, le pondré un radio de 3 metros.

Altura

Luego de otorgarle un radio, nos pedirá darle una altura, en este caso le daré una altura de 12 metros.

Resultado

Oprimiendo la tecla "shift" mas el "scroll" mientras se mueve el mouse, se puede ajustar la perspectiva del modelo o mesa de trabajo.
